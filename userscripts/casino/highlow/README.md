# Torn High-Low Brain

**What it does**
- Tracks the remaining deck (simple 4-per-rank model).
- Shows an overlay with probabilities and recommendation.
- Optional “Hide-worst” mode hides the lower-probability button.

**Hotkeys**
- `Alt+O` toggle overlay
- `Alt+H` toggle hide-worst
- `Alt+R` reset deck (new session)
- `Alt+M` toggle mode (profit/merit placeholder)

**Install**
- Primary: publish on Greasy Fork and install from there (auto-updates).
- Dev: Tampermonkey Dashboard → Add new script → paste file → Save.
